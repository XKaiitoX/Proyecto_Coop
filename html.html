Comando Explicacion
html 5 Es el estandar que sirve para definir la estrucutura y el contenido de una pagina web
H1 - H6 Estas etiquetas de título se emplean dentro de un sitio web para identificar los temas y subtemas de los que trata la página.
link especifica la relación entre el documento actual y un recurso externo
p es el apropiado para distribuir el texto en párrafos
id muestra el UID y el GID del usuario especificado, además de sus grupos secundarios
class se encarga de adaptar los datos de entrada para que puedan ser enviados como frames validos aceptados por el protocolo de la planta
div define las divisiones lógicas existentes en el contenido de una página web
span Sirve para aplicar estilo al texto o agrupar elementos en línea
a href el atributo href se integra perfectamente en tu página web y aparece ahí en forma de hipervínculo
img permite que se muestren las imágenes, pero en caso de que el archivo esté dañado también es capaz de describir el gráfico con algunos atributos
br en un determinado punto del texto equivale a presionar la tecla ENTER
hr representa un cambio de tema entre párrafos
table permiten acceder a conjuntos de filas
tr permite al Usuario definir explícitamente como estará compuesto el conjunto/td>
th efine una celda como encabezado de un grupo de celdas en una tabla
td define la celda de una tabla que contiene datos
thead define una sección de encabezado en una tabla
captation Es el encargado de darle un título descriptivo a las tablas.
tfoot define una sección de pie en una tabla
button se puede crear un botón de comando que abra otro formulario
title da el titulo de la pagina
